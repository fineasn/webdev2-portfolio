Social Media Learning

About this Website

This website contains information on how Social Media can be used as a learning platform nowadays. It goes over 5 apps, Instagram, Snapchat, Twitter, Whatsapp and Facebook. For each app we have included information on how to register for the app, how to use the app, a video showing how the app is used, cautions and warnings when using the app and the way they can be used for teaching and learning.
